# Weather App

### About App
This is a web app that displays The weather forecast for the next 5 days. The data source of the app is https://openweathermap.org/api.

### installation

> npm i

### run app

> npm run start

### bootstrap

> npm run bootstrap

### clean

> npm run clean

### test

> npm run test

### graphiql

> http://localhost:8080/graphql
